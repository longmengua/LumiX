# LumiX

`LumiX` 是交易所專案代號，名稱來自 `Lumi` 與 `X`。

`Lumi` 帶有光、照亮與清晰可見的意象，`X` 代表 `exchange`。

LumiX 的目標是建立一套可正式營運的線上交易所系統，涵蓋使用者介面、後端服務、交易核心、錢包、風控、對帳與營運支援。
