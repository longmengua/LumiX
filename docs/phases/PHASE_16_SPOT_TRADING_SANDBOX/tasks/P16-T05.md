## Verification

```text
BUY price >= SELL price 才可 match
BUY higher price priority
SELL lower price priority
same price time priority
matchedQuantity = min(buyRemaining, sellRemaining)
full fill => FILLED
partial fill => PARTIALLY_FILLED
no match => OPEN
trade price uses maker order price
```
