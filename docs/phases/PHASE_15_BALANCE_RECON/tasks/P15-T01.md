## Scope

```text
- 定義 trading runtime core scope gate
- 定義 safety contracts
- 建立最小 package skeleton / enum / policy
- 驗證仍沒有正式 money movement runtime
- 驗證 LedgerPostingCommandBoundary 尚未接 LedgerAppendOnlyJdbcAdapter
```

## Implementation rules

```text
- 不要接 LedgerPostingCommandBoundary 到 LedgerAppendOnlyJdbcAdapter
- 不要新增 LedgerPostingService
- 不要新增 BalanceMutationService
- 不要新增 ReservationRuntimeService
- 不要新增 SettlementService
- 不要新增 @Repository
- 不要新增 @Transactional
- 不要新增 DB client call
- 不要修改 migration
- 不要更新 balance_projections
- 不要改前端
- 不要宣稱 production-ready
```

## Verification

```text
所有 amount / price / quantity 必須 BigDecimal，不得 float / double
所有 money movement 必須 HUMAN_REVIEW_REQUIRED
requestId 不等於 idempotency guarantee
idempotency key 才能代表 duplicate prevention contract
ledger append 成功不代表 balance projection 已同步
balance projection 不可作為資金真相來源
reservation 不可直接改 ledger，必須經 application boundary
settlement 必須是 explicit process，不可由 matching 或 order runtime 偷寫 ledger
source-scan 確認沒有 LedgerPostingService、BalanceMutationService、ReservationRuntimeService、SettlementService、@Repository、@Transactional
source-scan 確認 LedgerPostingCommandBoundary 仍未引用 LedgerAppendOnlyJdbcAdapter
source-scan 確認沒有 balance_projections / reservation / settlement runtime SQL
P14 high-risk guardrail 測試仍通過
```

## Risk

```text
High。P15 一旦把 scope gate 寫成 runtime implementation，後面 balance / reservation / settlement 都會被誤接進正式交易路徑。
```
