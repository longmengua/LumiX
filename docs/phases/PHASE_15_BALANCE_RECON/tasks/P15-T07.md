## Verification

```text
ledger 是 source of truth
balance_projections 是 read model
reservation 是 hold/release 狀態模型，不是 ledger entry 替代品
settlement 是 explicit process，不能由 matching / order runtime 偷寫 ledger
reconciliation 必須比較 ledger_entries derived totals、balance_projections rows、reservation locked amounts、settlement expected movements
mismatch 必須產生 review / incident / repair flow，不得自動靜默修正
reconciliation 必須可追蹤 requestId，但 requestId 不是 idempotency guarantee
source-scan 確認 main source 沒有 ReconciliationService、ReservationRuntimeService、SettlementService、MatchingService、OrderPlacementService、FuturesService、LiquidationService、WithdrawalService、BalanceMutationService、BalanceProjectionService、LedgerPostingService、@Repository、@Transactional、JdbcTemplate、EntityManager
source-scan 確認沒有 INSERT / UPDATE / DELETE reservations、balance_projections、ledger_journals、ledger_entries、idempotency_keys、outbox_events、audit_logs、orders、trades runtime SQL
```

## Risk

```text
High。reconciliation 一旦被誤接成自動修正或寫回流程，就會把對帳路徑變成資金變動路徑。
```
