## Verification

```text
ledger 是 source of truth
balance_projections 是 read model
projection 可以 rebuild / replay / reconcile
projection lag 必須可觀測
projection mismatch 必須進 reconciliation flow
projection runtime 未來必須能從 ledger_entries 推導 total / available / locked
source-scan 確認沒有 BalanceProjectionService、BalanceMutationService、@Repository、@Transactional
source-scan 確認沒有 balance_projections insert / update / delete runtime SQL
source-scan 確認沒有 reservation hold / release runtime SQL
source-scan 確認沒有 settlement runtime SQL
```

## Risk

```text

```
