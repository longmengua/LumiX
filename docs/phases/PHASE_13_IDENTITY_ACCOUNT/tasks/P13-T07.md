## Scope

```text
- 新增 security boundary 文件
- 新增 auth / permission / principal / api key / admin guard 類 package skeleton
- 新增 marker interface、enum、policy skeleton
- 新增測試，驗證高風險 security policy 分類
- 更新 backend-module-boundary / error-response-boundary / api-versioning 文件
```

## Implementation rules

- 不要實作登入流程。
- 不要實作 API key runtime validation。
- 不要實作 withdrawal signing / broadcast。
- 不要實作 ledger posting。
- 不要實作 admin operation runtime。
- 不要實作 settlement / risk override runtime。
- 不要新增或修改 Flyway migration。
- 不要改前端。
- 所有新增 public class / public method 必須有繁體中文維護性註解。
