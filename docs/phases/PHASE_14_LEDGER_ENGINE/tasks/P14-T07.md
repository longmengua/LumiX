
## Verification

```text
PostgreSQL version: 16
Replay V001 to V008
Seed users / accounts / assets / account_assets
驗證 ledger_journals +1 與 ledger_entries +2
驗證 entry insert failure rollback
驗證 balance_projections row count 不變
驗證 posting boundary 尚未接 DB adapter
```

## Risk

```text
High。PostgreSQL replay 會直接暴露 migration 與 adapter 行為差異，任何錯誤都不能被視為 H2 一定成立。
```
