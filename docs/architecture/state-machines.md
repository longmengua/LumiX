# 狀態機

## 訂單狀態機

```text
+---------+
| 新建    |
+----+----+
     |
     v
+----+----+       reject        +----------+
| 已接受  | ----------------> | 已拒絕   |
+----+----+                    +----------+
     |
     +------ 部分成交 --------+
     |                        v
     |                  +-----+------+
|                  | 部分成交中 |
     |                  +-----+------+
     |                        |
     |                        +---- 補足剩餘 -----------+
     |                        |                        v
     v                        |                  +-----+----+
+----+----+                   +----------------> | 已成交   |
| 開啟    |                                      +----------+
+----+----+
     |
     +---- 取消 -----> +-----------+
                        | 已取消    |
                        +-----------+
```

## 預留狀態機

```text
+-----------+
| 已請求    |
+-----+-----+
      |
      v
+-----+-----+      fail       +--------+
| 已凍結    | --------------> | 失敗   |
+--+-----+--+                 +--------+
   |     |
   |     +---- 釋放 ----> +----------+
   |                       | 已釋放   |
   |                         +----------+
   |
   +---- 扣款 -----------> +----------+
                             | 已扣款   |
                             +----------+
```

## 提款狀態機

```text
已請求 -> 風控審查 -> 已核准 -> 簽章中 -> 已廣播 -> 已確認
     |            |             |          |            |
     v            v             v          v            v
  已拒絕       已拒絕        已取消      失敗         失敗
```
