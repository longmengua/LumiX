# 部署視圖

```text
+---------------------+          +---------------------+
| CDN / 邊緣          | -------> | 網頁應用程式        |
+---------------------+          +---------------------+
                                          |
                                          v
+---------------------+          +---------------------+
| 負載平衡器          | -------> | API Pod             |
+---------------------+          +---------------------+
                                          |
                  +-----------------------+-----------------------+
                  |                       |                       |
                  v                       v                       v
        +------------------+    +------------------+    +------------------+
| Worker Pod       |    | PostgreSQL       |    | Redis            |
        +------------------+    +------------------+    +------------------+
                  |
                  v
        +------------------+    +------------------+
| 鏈節點           |    | 簽章邊界        |
        +------------------+    +------------------+
```

## 部署備註

- API Pod 是無狀態的。
- PostgreSQL 是關鍵的有狀態依賴。
- Redis 可替換，且不得保存唯一的資金真相。
- 簽章邊界應與公開 API 路徑隔離。
- 鏈上監聽器必須具備冪等與可重放特性。
