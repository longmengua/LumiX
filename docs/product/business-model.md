# Business Model

LumiX 的收入不只是交易畫面，而是完整交易與資產服務能力。

## Revenue streams

```text
Trading fees
  - maker fee
  - taker fee
  - VIP tier discount

Withdrawal fees
  - network fee pass-through
  - optional service fee

Listing / market operations
  - future listing process
  - market maker support

Enterprise / API
  - future premium API or institutional access
```

## Revenue must be auditable

```text
Trade matched
  -> fee calculated
  -> fee ledger entry posted
  -> revenue account balance updated
  -> report export available
```

若費用不能被 ledger 化，收入就不能被可信地審核。
