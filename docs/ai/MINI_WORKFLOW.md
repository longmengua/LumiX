# Mini Workflow

```text
+------------+     +------------+     +------------+     +------------+
| Read task  | --> | Implement  | --> | Verify     | --> | Summarize  |
+------------+     +------------+     +------------+     +------------+
       ^                                                     |
       |                                                     v
       +-------------------- next task after review ----------+
```
