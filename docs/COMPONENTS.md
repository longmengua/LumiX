# LumiX 元件索引

## 版面元件

- `AppLayout`：整體頁面骨架
- `Header`：頂部導覽列
- `Sidebar`：側邊導覽欄
- `PageHeader`：頁面標題區

## 基礎元件

- `Card`：資訊卡片
- `Badge`：狀態標籤
- `LoadingState`：載入中
- `EmptyState`：空狀態
- `ErrorState`：錯誤狀態
- `ConfirmDialog`：確認對話框

## 交易輔助元件

- `AmountText`：數量顯示
- `PriceText`：價格顯示
- `PnlText`：損益顯示
- `RiskRatioBar`：風險比率條
