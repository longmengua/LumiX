# Risk Control

## Risk checkpoints

```text
registration
login
deposit credit
order placement
order cancellation burst
withdrawal request
withdrawal approval
admin action
market halt / resume
```

## Controls

- Rate limits.
- Velocity limits.
- Withdrawal amount limits.
- Address allowlist / cooldown if enabled.
- Market halt.
- Asset withdrawal pause.
- Manual review queue.

## Rule

Risk rejection must be explicit and auditable. Silent balance mutation is prohibited.
