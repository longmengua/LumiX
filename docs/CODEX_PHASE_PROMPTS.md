# Codex 章節提示索引

這份文件只做簡單索引，方便快速找每一章的實作方向。每個章節的完整提示都放在自己的 Phase 文件末尾。

## 使用原則

- 每次只做一個章節。
- 不跳章。
- 不把 stub、interface、mock、placeholder 或 TODO 當完成。
- 完成後先進入人工審核。
- 沒有 Phase 36 的完整門檻，不要宣稱正式上線就緒。
