# LumiX Routes

## Public routes

- `/` - Home
- `/markets` - Markets
- `/spot/:symbol` - Spot trading placeholder
- `/futures/:symbol` - Futures trading placeholder
- `/margin/:symbol` - Margin trading placeholder

## Account routes

- `/assets` - Assets
- `/orders` - Orders
- `/positions` - Positions
- `/account` - Account overview

## Admin routes

- `/admin` - Admin dashboard placeholder
